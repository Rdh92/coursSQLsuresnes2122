INSERT, UPDATE et DELETE

1- INSERT INTO, insertion

  • INSERT INTO employes (prenom, nom, sexe, service, date_embauche, salaire) VALUES ('Alexis', 'Richard','m','informatique', '2011-12-28', 1800);
  • Afficher cet enregistrement on le cherche par le nom de famille : SELECT * FROM employes WHERE nom = 'Richard';ou bien on cherche le dernier enregistrement SELECT * FROM employes ORDER BY id_employes DESC LIMIT 1;
  • INSERT INTO employes VALUES (NULL, 'Jeanne', 'Richy', 'f', 'informatique', '2021-01-28', 1800);
  • NULL nous permet de laisser le champ id_employes se remplir seul car il est auto-incrémenté

    • 2- UPDATE, requête de mise à jour

  • Augmenter le salaire de l'employée, Jeanne, dont l'id est 993UPDATE SET salaire = 1900 WHERE id_employes = 992;
  • On utilise toujours des ID pour mettre à jour les informations de la BDD car c'est le seul élément unique
  • Pour faire plusieurs mises à jour en même temps, il faudra respecter l'ordre des colonnes et les séparer par une virgule :UPDATE employes SET service = 'commercial', salaire = 2000 WHERE id_employes = 993;

    • 3- DELETE, requête de suppression

    • Supprimer un enregistrement avec DELETE : DELETE FROM employes WHERE id_employes = 388;
    • Supprimer plusieurs enregistrements, ici on supprime toutes les personnes du service informatique :DELETE FROM employes WHERE service = 'informatique';
